# AI Guide

A SwiftUI iOS app that provides in-depth, interactive guides for AI, homelab, and self-hosting topics.

## Features

- Step-by-step guides for local AI, cloud AI, virtualization, and networking
- Searchable and easy to navigate
- Clean, modern SwiftUI interface
- Organized by category: Local AI, Cloud AI, Virtualization, Networking/Remote Access

## Requirements

- iOS 17+
- Xcode 15+
- Swift 5.9+

## Installation

1. Clone the repo
2. Open in Xcode
3. Build and run on simulator or device

## Usage

Browse guides by category, follow step-by-step instructions, and reference troubleshooting sections when needed.
